# Screenshots Directory

## How to Add Screenshots

1. **Take screenshots** of your running application:
   - Form Builder interface (`form-builder.png`)
   - Form Preview with conditional logic (`form-preview.png`) 
   - Response management dashboard (`responses.png`)

2. **Recommended screenshot tools:**
   - macOS: `Cmd + Shift + 4` for selective screenshots
   - Windows: Snipping Tool or `Win + Shift + S`
   - Browser extensions: Awesome Screenshot, Full Page Screen Capture

3. **Screenshot specifications:**
   - Format: PNG or JPG
   - Resolution: At least 1200px wide
   - Show the key features and UI elements clearly

4. **Required screenshots:**
   - `form-builder.png` - Show the form creation interface with Airtable fields
   - `form-preview.png` - Show form with conditional logic in action
   - `responses.png` - Show the responses dashboard with submitted data

## Demo Video

Create a 2-3 minute demo video showing:
1. Creating a form with Airtable fields
2. Setting up conditional logic
3. Filling and submitting the form  
4. Viewing responses in the dashboard
5. Verification of data in both MongoDB and Airtable

Save as `demo-video.mp4` in the docs folder.
